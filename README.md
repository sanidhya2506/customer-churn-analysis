# 📊 Customer Churn Prediction & Segmentation

## 📌 Overview

This project focuses on analyzing customer behavior and predicting churn using machine learning.  
It also includes **customer segmentation** to identify different user groups and provide actionable business insights.

---

## 🌐 Live Demo

Check out the **interactive Streamlit app** here:  
[Customer Segmentation & Churn Prediction App](https://customer-churn-analysis-ulayncrvzpxtidhcmdv46g.streamlit.app/)

---

## 🎯 Objectives

- Predict whether a customer will churn or not
- Identify key factors influencing churn
- Segment customers into meaningful groups
- Provide insights for improving customer retention

---

## 📂 Dataset

- **Telco Customer Churn Dataset**
- Includes features like:
  - Demographics (gender, senior citizen)
  - Account information (tenure, contract type)
  - Billing details (monthly charges, payment method)

---

## ⚙️ Tech Stack

- Python 🐍
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Streamlit (for interactive deployment)

---

## 🔍 Workflow

### 1. Data Preprocessing
- Handled missing values
- Encoded categorical variables
- Removed unnecessary columns

### 2. Exploratory Data Analysis (EDA)
- Analyzed customer behavior
- Visualized churn patterns

### 3. Model Building
- Trained a Random Forest Classifier
- Achieved ~79% accuracy

### 4. Model Evaluation
- Used classification report metrics:
  - Precision
  - Recall
  - F1-score

### 5. Feature Importance
- Identified key drivers of churn:
  - Contract type
  - Tenure
  - Monthly charges

### 6. Customer Segmentation
- Applied KMeans clustering
- Grouped customers based on behavior
- Identified high-value and at-risk segments

---

## 📊 Results

- Built an end-to-end churn prediction system
- Extracted meaningful business insights
- Segmented customers for targeted strategies

---

## 💡 Business Insights

- Customers with short tenure are more likely to churn
- Monthly charges significantly impact churn behavior
- Long-term contracts reduce churn probability
